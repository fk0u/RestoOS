# RestoOS Icon Asset Pack

Folder ini untuk icon PNG/JPG agar tampilan shell RestoOS terasa seperti OS production.

## Nama file yang dikenali
- `browser.png`
- `mail.png`
- `pos.png`
- `settings.png`
- `notepad.png`
- `terminal.png`
- `designer.png`
- `designer-frame.png`
- `netbeans.png`
- `explorer.png`
- `control.png`
- `kitchen.png`
- `monitor.png`
- `admin.png`
- `user.png`

## Catatan
- Ukuran direkomendasikan `64x64` atau `128x128` transparan.
- Asset akan otomatis dipakai jika file ditemukan.
- Jika file belum ada, aplikasi memakai vector fallback dari `IconAssets` agar tetap konsisten.
